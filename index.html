Dentro del Box Model

El padding el tamaño que tienen los 4 lados de un box model, es decir es la primera capa que envuelve al contenedor, le podemos dar un tamaño a cada lado, o tambien podemos poner uno solo que aplica a los 4 lados en este caso le pusimos 20px. Si le pones 10 px 20px decimos que tendra 10 px a los lados superior e inferior, y 20 px a los lados derecha e izquierda. si pongo 10px 20px 5px 8px si es a cada lado

El concepto de border (borde) en CSS se refiere a la propiedad que permite agregar un borde alrededor de un elemento HTML. Este borde se representa visualmente como una línea que rodea el contenido del elemento. La propiedad border se puede utilizar para especificar el ancho, el estilo y el color del borde.

"margin" se refiere al espacio exterior alrededor de un elemento HTML. La propiedad margin se utiliza para establecer el margen de separación entre los bordes del elemento y los elementos adyacentes. Puede aplicarse a los cuatro lados del elemento (arriba, derecha, abajo e izquierda) o a cada lado de forma individual.

Segundo Box Model

El selector universal (*) en CSS se utiliza para aplicar estilos a todos los elementos en una página web. Cuando se aplica al contexto del modelo de caja (box model), puede ser útil en ciertos casos para normalizar el espaciado y dimensiones predeterminadas de todos los elementos en una página.

Los márgenes negativos en CSS permiten que un elemento se superponga a otros elementos cercanos o incluso a su contenedor principal. Al asignar un margen negativo a un elemento, estás moviendo sus bordes más allá de la posición normal que ocuparía según el modelo de caja.

La regla principal para entender el colapso de márgenes verticales es que solo el margen más grande entre dos márgenes adyacentes prevalecerá, mientras que el margen más pequeño se ignorará.
